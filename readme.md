# TextEditor

## Getting Started
The goal of the project is to create a text editor with basic functions using algorithms, data structures and what we learned during the API course.
## Documentation

The project is made using only C language.

### Requirements

* change: (ind1,ind2)c
* delete: (ind1,ind2)d
* print: (ind1,ind2)p
* undo: (num)u
* redo: (num)4


### Built With

* [CLion](https://www.jetbrains.com/clion/) - IDE
* [Valgrind](https://www.valgrind.org/) - Instrumentation framework
